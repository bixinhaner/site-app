📝 技术说明

前端技术栈： UniApp + Vue3 + Pinia + UniUI
后端技术栈： FastAPI + SQLAlchemy + PostgreSQL + JWT
部署方案： Docker + Nginx + SSL证书
开发工具： VS Code + HBuilderX + Git + Claude Code

该系统采用前后端分离架构，支持多端部署（Android/iOS/H5），具备完整的用户权限管理、 离线数据同步、GPS定位验证、照片水印处理等企业级功能特性。
