## ❓ 故障排除

### 问题1: 后端启动失败
```bash
# 检查端口占用
lsof -i :8000

# 强制停止占用进程
kill -9 <PID>
```

### 问题2: 测试脚本失败
**可能原因**：
- 后端服务未启动
- 端口配置不正确
- 测试账号密码错误

**解决方法**：
- 确认后端在8000端口运行
- 检查API_BASE_URL配置
- 使用正确的测试账号（inspector/inspector123）

### 问题3: App端仍看不到描述
**可能原因**：
- 前端缓存
- 后端未重启
- 检查项本身没有描述

**解决方法**：
1. 下拉刷新页面
2. 确认后端已重启
3. 运行测试脚本验证API
4. 检查控制台是否有错误
