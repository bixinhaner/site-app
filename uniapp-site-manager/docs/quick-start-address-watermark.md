您的水印功能已经自动升级！现在拍照时会自动显示详细地址信息而不仅仅是坐标。

## 🔧 获得更准确的地址（可选）

### 方案1：配置高德地图API（推荐）
1. 访问 https://lbs.amap.com/ 注册账号
2. 创建应用，选择"Web服务API"
3. 获取API Key
4. 在代码中配置：

```javascript
// 在 config/geocoding.js 中
export const geocodingConfig = {
  amap: {
    apiKey: 'your-amap-api-key-here', // 填入您的API Key
    enabled: false
  }
}
```

### 方案2：配置腾讯地图API
1. 访问 https://lbs.qq.com/ 注册账号
2. 获取API Key
3. 配置到 `geocodingConfig.tencent.apiKey`

### 方案3：配置百度地图API
1. 访问 https://lbsyun.baidu.com/ 注册账号
2. 获取AK
3. 配置到 `geocodingConfig.baidu.ak`

## ⚠️ 注意事项

### 位置权限
- 首次使用需要授权GPS定位权限
- 确保在网络连接良好的环境下使用

### 网络环境
- 地址获取需要网络连接
- 如果网络不好，会显示基本GPS坐标
- 系统有多重降级保障，不会影响基本拍照功能

### 精度建议
- 在户外环境下GPS精度更高
- 室内可能只能获取大概位置
- 系统会显示定位精度供参考

## 🛟 故障排除

### 问题1：地址显示"获取中..."
**原因**：网络连接问题或API调用失败
**解决**：
- 检查网络连接
- 等待几秒钟重试
- 系统会自动降级显示坐标

### 问题2：地址不够详细
**原因**：使用的是OpenStreetMap免费服务
**解决**：配置商用地图API密钥（高德/腾讯/百度）

### 问题3：GPS获取失败
**原因**：位置权限未授权或GPS信号差
**解决**：
- 检查应用位置权限
- 移动到信号较好的地方
- 系统会使用基本水印模式
